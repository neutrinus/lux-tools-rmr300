# Harmonogram koszenia — konfiguracja

Źródło: `doc/manual.pdf` (instrukcja oryginalna PL, str. 87–88)

## Panel sterowania

Przyciski: `[START]` `[HOME]` `[OK]` oraz przełącznik zasilania `[ ]`.

## 1. Ustawienie roku, daty i godziny (wymagane dla harmonogramu)

1. Przytrzymaj `[START]` przez **5 sekund** (aż symbol `[ ]` i liczba zaczną migać)
2. Ustaw **rok** → `[START]`/`[HOME]` wybór, `[OK]` potwierdź
3. Ustaw **datę** (MM.DD) → `[START]`/`[HOME]` wybór, `[OK]` potwierdź
4. Ustaw **godzinę** (HH:MM) → `[START]`/`[HOME]` wybór, `[OK]` potwierdź

Po zakończeniu wyświetlacz pokaże `[IDLE]`.

## 2. Zmiana godziny startu (domyślnie 9:00)

1. Przytrzymaj `[START]` + `[OK]` jednocześnie przez **3 sekundy**
2. Gdy miga `[0-:--]`, ustaw **godzinę** → `[START]`/`[HOME]` wybór, `[OK]` potwierdź
3. Następnie ustaw **minuty** → `[START]`/`[HOME]` wybór, `[OK]` potwierdź
4. Wyświetlacz pokaże `[IDLE]` + sygnał dźwiękowy — kosiarka zacznie kosić o nowej godzinie

## 3. Zmiana dziennego czasu koszenia (domyślnie 3h)

1. Przytrzymaj `[OK]` przez **3 sekundy**
2. Gdy miga `[06 H]`, wybierz **1–24 h** → `[START]`/`[HOME]` wybór, `[OK]` potwierdź

## 4. Zmiana dni koszenia w tygodniu (domyślnie 5 dni)

1. Przytrzymaj `[HOME]` + `[OK]` jednocześnie przez **3 sekundy**
2. Gdy miga `[-05-]`, wybierz **3 lub 7 dni** → `[START]`/`[HOME]` wybór, `[OK]` potwierdź

## 5. Czujnik deszczu (domyślnie włączony)

1. Przytrzymaj `[HOME]` przez **3 sekundy** (pojawi się `[RAIN]`)
2. Naciśnij `[OK]`, przełącz `[ON]`/`[OFF]` → `[START]`/`[HOME]` wybór, `[OK]` potwierdź

## Uwagi

- Kosiarka automatycznie wraca do stacji ładowania gdy poziom baterii jest niski
- Po naładowaniu wznawia koszenie lub czeka zgodnie z harmonogramem
- Krawędzie są koszone automatycznie w każdy **wtorek i piątek**
- Ustawienie daty/godziny jest **obowiązkowe** — bez tego harmonogram nie działa poprawnie
